# 自定义Android - Magisk模块 
功能：
1. Android全局使用定制字体，可根据语言文字使用不同字体（system/etc/fonts.xml中配置）
以及参考[killgmsfont](https://github.com/MrCarb0n/killgmsfont)模块禁用GMS的字体服务（用于Google应用）
2. Android 6启动动画；
3. 定时删除垃圾文件；
4. 禁用OEM加密（允许播放DRM保护内容）；
